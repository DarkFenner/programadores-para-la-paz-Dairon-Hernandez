El comando uname -a muestra información detallada del sistema operativo, incluyendo 
el nombre del sistema, la versión del kernel, la arquitectura del procesador y otros 
datos técnicos del equipo.

El comando whoami muestra el nombre del usuario que actualmente tiene iniciada la sesión en el sistema.

Conocer esta información es útil al administrar un servidor porque permite 
identificar el sistema operativo que se está utilizando, verificar la versión 
instalada y confirmar con qué usuario se están ejecutando las tareas. Esto ayuda a 
realizar configuraciones, solucionar problemas y aplicar medidas de seguridad de 
manera adecuada.
